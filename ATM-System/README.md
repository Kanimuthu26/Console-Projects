# ATM System (Console-Based)

## Overview
This is a console-based ATM simulation developed using Java. It demonstrates basic banking operations such as authentication, balance inquiry, deposit, and withdrawal.

## Features
- PIN-based user authentication
- Balance checking
- Deposit money
- Withdraw money with validation
- Continuous menu-driven interaction

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner for input handling

## Project Structure
- `Account.java` → Handles account data and operations
- `ATM.java` → Controls user interaction and transaction flow
- `Main.java` → Entry point of the application

## How to Run
1. Navigate to `src` folder
2. Compile: javac*.java
3. Run: java Main


## Key Concepts Demonstrated
- Encapsulation (Account class)
- Separation of concerns
- Basic input validation

## Future Enhancements
- Multiple account support
