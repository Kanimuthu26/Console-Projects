# Console-Based Java Projects

## Overview
This repository contains three console-based Java applications designed to demonstrate core programming concepts, object-oriented design, and basic business logic implementation.

## Projects Included

### 1. ATM System
Simulates banking operations such as authentication, balance checking, deposit, and withdrawal.

### 2. Library Management System
Manages library operations including adding books, issuing, and returning.

### 3. Supermarket Billing System
Implements a real-world billing system with cart management, credit handling, and reward logic.

## Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Collections Framework

## How to Run
For each project:
1. Navigate to the respective `src` folder
2. Compile: javac *.java
3. Run: java Main


## Key Highlights
- Modular project structure
- Clean separation of concerns
- Real-world problem modeling
- Scalable design approach
