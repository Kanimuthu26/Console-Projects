# Library Management System (Console-Based)

## Overview
A console-based application that manages library operations such as adding books, issuing books, and returning books.

## Features
- Add new books
- View all books
- Issue books (availability check)
- Return books
- Track book status (Issued/Available)

## Technologies Used
- Java
- Collections Framework (ArrayList)
- OOP principles

## Project Structure
- `Book.java` → Represents book entity
- `Library.java` → Contains core business logic
- `Main.java` → Handles user interaction

## How to Run
1. Navigate to `src` folder
2. Compile: javac *.java
3. Run: java Main


## Key Concepts Demonstrated
- List data structure usage
- State management (issued vs available)
- Modular design
